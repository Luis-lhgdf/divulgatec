Glossário de Tags e Formatações CSS - DivulgaTec

Este glossário serve como um guia de referência para as tags HTML e formatações CSS utilizadas no desenvolvimento do site DivulgaTec. Aqui você encontrará informações detalhadas sobre as estruturas e estilos aplicados no projeto.

Estrutura HTML do Site:

Tags e Atributos:

Estudado em sala de aula:
  • html:

    Tag principal que envolve todas as outras tag e conteúdo do nosso site.

  • head:

    Contém informações sobre a página, como título e links para estilos e possiveis configurações.

  • title:

    Define o título da página.

  • body:

    Contém todo o conteúdo visível da página.

  • form:

    Define um formulário HTML.

  • label:

    Descreve um campo do formulário, como "Nome" ou "Email"

  • textarea:

    Campo de texto para formulário.

  • input:

    Campo de entrada de dados para formulário.

  • action:

    Atributo para especificar o URL de destino para o envio dos dados do formulário.

  • type:

    Atributo para especificar o tipo do campo do input (text, email, etc.).

  • name:

    Atributo para o nome do campo e variavel.

  • placeholder:

    Atributo para inserir um texto de orientação para o usuário.

  • tr:

    Linha de tabela.

  • td:

    Célula de tabela.

  • fieldset:

    Agrupam informações criando um caixa e add um título para esse grupo.

  • legend:

    Fornece um título ou legenda para o agrupamento feito pela tag "fieldset".

  • img:

    Imagem.

  • center:

    Tag para centralizar conteúdo.

  • strong:

    Deixa o texto em negrito para dar ênfase..

  • table:

    Cria uma Tabela.

  • h1, h2, h3:

    Títulos de cabeçalho.

  • p:

    Parágrafo.

  • br:

    Quebra de linha.

  • hr:

    Adiciona uma linha horizontal.

  • a:

    Link de âncora.

  • ul:

    Lista não ordenada.

  • ol:

    Lista ordenada.

  • li:

    Item de lista.

  • u:

    usada para sublinhar um texto.

  • i:

    usada para deixar um texto em italico.

  • rows:

    Atributo para especificar o número de linhas visíveis.

  • href:

    Atributo de URL do link.

  • target="_blank":

    Atributo para abrir o link em uma nova aba/janela.

Iniciativa de aprendizado próprio:
  • DOCTYPE html:

    Define o tipo de documento HTML no inicio da pagina.

  • meta:

    Tags meta para configuração do documento.

  • link:

    Tag para vincular um arquivo CSS externo.

  • header:

    Define o cabeçalho e a seção de cabeçalho da pagina.

  • section:

    Cria uma seção que pode ser personalizada.

  • div:

    Cria uma seção ou bloco para agrupar conteúdo do site.

  • main:

    Define o conteúdo e as informações principais do nosso site.

  • footer:

    Rodapé da nossa página.

  • content="width=device-width, initial-scale=1.0":

    Atributo para Ajusta a página para dispositivos móveis (padrao do vscode).

  • rel="stylesheet":

    Atributo que especifica a relação entre a pagina html e o css.

  • lang="en":

    Atributo que especifica o idioma da pagina.

  • charset:

    Atributo que define o conjunto de caracteres aceito na nossa pagina.

Propriedades CSS Utilizadas no site:

Propriedades CSS:

Seletor Universal
  • *

    Define um estilo para todos os elementos da nossa pagina.

IDs e Classes
  • ID (#):

    Identifica uma tag específica para estilização e tambem usado para nossa barra de navegação.

  • Classes (.):

    Utilizamos as classes para criar estilos reutilizáveis para várias tag ao mesmo tempo.

Unidades de Medida
  • rem:

    Unidade relativa ao tamanho da fonte

  • px:

    Unidade de medida fixa, um pixel é uma única ponto em uma imagem digital.

  • %:

    Porcentagem para largura e altura em layouts responsivos do nosso site.

  • vh:

    É uma medida que representa 1% da altura total da tela do dispositivo.

  • auto:

    Valor automático, usamos para alinhar e dimensionar elementos automaticamente.

Pseudo-classes
  • :active

    Estilos para quando o link está ativo ou o cursor passa sobre ele.

  • :hover

    Estilização quando o cursor está sobre o link do botao.

Propriedades e Valores
  • align-items:

    Alinha os itens no meio do contêiner que criamos.

  • background-color:

    Define a cor de fundo.

  • background-image:

    Define uma imagem de fundo.

  • background-position:

    Centraliza a posição da imagem de fundo.

  • background-repeat:

    Evita que a imagem de fundo se repita.

  • background-size:

    Ajusta o tamanho da imagem de fundo.

  • border-radius:

    Arredonda os cantos do elemento, como botoes criados com as div.

  • color:

    Define uma cor do texto.

  • display:

    Controla como o elemento é exibido na tela.

  • flex:

    Controla a flexibilidade do elemento em um contêiner que criamos personalizado.

  • flex-direction:

    Muda a direção dos itens para vertical.

  • font-family:

    Define a fonte utilizada ao longo do nosso site.

  • font-size:

    Define o tamanho da fonte.

  • font-weight:

    Define o peso da fonte do nosso site como semibold ou negrito.

  • gap:

    Define o espaçamento entre os itens do contêiner flex que criamos.

  • height:

    Define a altura do elemento.

  • justify-content:

    Distribui o espaço restante entre os itens em um contêiner flex.

  • left:

    Move o elemento para a esquerda.

  • line-height:

    Define a altura das linhas de texto.

  • list-style:

    Remove os marcadores de lista ul.

  • margin:

    Define as margens externas do elemento.

  • margin-bottom:

    Define a margem inferior do elemento.

  • margin-top:

    Define a margem superior do elemento.

  • max-width:

    Define a largura máxima do elemento.

  • min-height:

    Define a altura mínima do elemento.

  • padding:

    Adiciona espaço interno ao redor do conteúdo do elemento para melhorar a distancia dos textos.

  • position:

    Define a posição do elemento na nossa página.

  • text-align:

    Define a posição do texto.

  • text-decoration:

    Remove a decoração de texto, como sublinhado.

  • top:

    Define a posição superior do elemento.

  • width:

    Define a largura do elemento.

  • z-index:

    Define a ordem em que os elementos são empilhados uns sobre os outros, usamos na nossa barra de navegação.
