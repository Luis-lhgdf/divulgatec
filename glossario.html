Glossário de Tags e Formatações CSS - DivulgaTec

Este glossário serve como um guia de referência para as tags HTML e formatações CSS utilizadas no desenvolvimento do site DivulgaTec. Aqui você encontrará informações detalhadas sobre as estruturas e estilos aplicados no projeto.

Estrutura HTML do Site:

Tags e Atributos:

Estudado em sala de aula:
  • html:

    Tag raiz que envolve todo o conteúdo HTML.

  • head:

    Cabeçalho do documento que contém metadados e links para arquivos externos.

  • title:

    Define o título da página.

  • body:

    Contém todo o conteúdo visível da página.

  • form:

    Define um formulário HTML.

  • label:

    Etiqueta descritiva para um campo de formulário.

  • textarea:

    Campo de texto multi-linha para formulário.

  • input:

    Campo de entrada para formulário.

  • action:

    Atributo para especificar o URL de destino para o envio dos dados do formulário.

  • type:

    Atributo para especificar o tipo do campo (text, email, etc.).

  • name:

    Atributo para o nome do campo e variavel.

  • placeholder:

    Atributo para inserir um texto de orientação para o usuário.

  • a:

    Link de âncora.

  • ul:

    Lista não ordenada.

  • ol:

    Lista ordenada.

  • li:

    Item de lista.

  • img:

    Imagem.

  • center:

    Tag para centralizar conteúdo.

  • strong:

    Texto forte ou enfatizado.

  • table:

    Tabela.

  • tr:

    Linha de tabela.

  • td:

    Célula de tabela.

  • fieldset:

    Agrupa elementos de formulário relacionados em um contêiner.

  • legend:

    Fornece um título ou legenda para o agrupamento feito pela tag "fieldset".

  • h1, h2, h3:

    Títulos de cabeçalho.

  • p:

    Parágrafo.

  • div:

    Divisão genérica ou contêiner.

  • br:

    Quebra de linha.

  • rows:

    Atributo para especificar o número de linhas visíveis.

  • href:

    Atributo de URL do link.

  • target="_blank":

    Atributo para abrir o link em uma nova aba/janela.

  • hr:

    Linha horizontal.

Iniciativa de aprendizado próprio:
  • DOCTYPE html:

    Declaração do tipo de documento.

  • meta:

    Tags meta para configuração do documento.

  • link:

    Tag para vincular um arquivo CSS externo.

  • section:

    Define uma seção genérica em um documento HTML.

  • header:

    Define o cabeçalho ou a seção de cabeçalho de um documento ou seção.

  • main:

    Define o conteúdo principal ou a parte central do conteúdo de um documento ou aplicativo.

  • footer:

    Rodapé da página.

  • charset:

    Atributo para definir o conjunto de caracteres do documento.

  • content="width=device-width, initial-scale=1.0":

    Atributo para Definir a largura da viewport e a escala inicial para dispositivos móveis.

  • rel="stylesheet":

    Atributo que especifica a relação entre o documento e a folha de estilo.

  • lang="en":

    Atributo que especifica o idioma do conteúdo.

Propriedades CSS Utilizadas no site:

Propriedades CSS:

Seletor Universal
  • *

    Resetando padding e margin em todos os elementos.

IDs e Classes
  • ID (#):

    Um ID é um seletor único usado para identificar um elemento específico em uma página HTML.

  • Classes (.):

    Uma classe é um seletor reutilizável usado para estilizar múltiplos elementos.

Unidades de Medida
  • rem:

    Unidade relativa ao tamanho da fonte do elemento raiz (<html>).

  • px:

    Unidade de medida fixa. Um pixel é uma única ponto em uma imagem digital.

  • %:

    Porcentagem de uma unidade pai. É usado principalmente para largura e altura em layouts responsivos.

  • vh:

    Viewport Height. Equivale a 1% da altura da viewport.

  • auto:

    Valor automático. Geralmente usado para alinhar ou dimensionar elementos automaticamente.

Pseudo-classes
  • :active

    Estilização quando o item está ativo.

  • :hover

    Estilização quando o cursor está sobre o item.

Propriedades e Valores
  • align-items:

    Centraliza os itens ao longo do eixo transversal em um contêiner flex.

  • background-color:

    Define a cor de fundo como branco.

  • background-image:

    Define uma imagem de fundo para o elemento.

  • background-position:

    Centraliza a posição da imagem de fundo.

  • background-repeat:

    Evita que a imagem de fundo se repita.

  • background-size:

    Ajusta a imagem de fundo para cobrir todo o elemento sem distorção.

  • border-radius:

    Arredonda os cantos do elemento.

  • color:

    Define a cor do texto como preto.

  • display:

    Controla como o elemento é exibido.

  • flex:

    Define o fator de flexibilidade para o elemento em um contêiner flex.

  • flex-direction:

    Define a direção do eixo principal em um contêiner flex como vertical.

  • font-family:

    Define a fonte utilizada para o texto.

  • font-size:

    Define o tamanho da fonte como 62.5% do tamanho padrão do navegador.

  • font-weight:

    Define o peso da fonte como semibold.

  • gap:

    Define o espaçamento entre os itens de um contêiner flex.

  • height:

    Define a altura do elemento.

  • justify-content:

    Distribui o espaço restante entre os itens em um contêiner flex.

  • left:

    Define a posição à esquerda do elemento.

  • line-height:

    Define a altura da linha do texto.

  • list-style:

    Remove os marcadores de lista padrão.

  • margin:

    Define as margens do elemento.

  • margin-bottom:

    Define a margem inferior do elemento.

  • margin-top:

    Define a margem superior do elemento.

  • max-width:

    Define a largura máxima do elemento.

  • min-height:

    Define a altura mínima do elemento.

  • padding:

    Define o preenchimento interno do elemento.

  • position:

    Define a posição do elemento como fixa, relativa ao contêiner.

  • text-align:

    Define a posição do texto a esquerda.

  • text-decoration:

    Remove a decoração de texto, como sublinhado.

  • text-transform:

    Transformação de texto para maiúsculas.

  • top:

    Define a posição superior do elemento.

  • width:

    Define a largura do elemento.

  • z-index:

    Define a ordem de empilhamento do elemento.
